Täckningskarta samtal via mobilnätet

Här kan du se Tele2s täckning för samtal via mobilnätet. Observera att täckningen kan påverkas av omgivningen, såsom berg och dalar eller att du befinner dig inomhus.

Täckningen på kartan är baserad på en estimerad utomhustäckning utfrån Tele2s basstationer och master i områdena. Observera att täckningen kan påverkas av omgivningen, såsom berg och dalar, kraftig vegetation eller om du befinner dig i eller vid byggnader. Olika typer av enheter (mobiltelefon, läsplatta, modem etc) samt modeller kan bara koppla upp på vissa våglängder av nätet, vilket kan påverka den upplevda kvaliteten.

Tele2 Fixit

Tele2 FixIt gör att du enkelt kan tala om för Tele2 när och var du har problem med mobilnätet. Med några enkla knapptryck skickar du information om den platsen du vill få bättre mottagning på. Genom din rapport får vi värdefull statistik så att vi hela tiden kan förbättra nätet.
