Orderstatus

Här kan du följa din order. Ange ditt ordernummer och ditt personnummer eller organisationsnummer nedan för att se status.

Gör så här

  1. Fyll i ordernummer som du fick via e-post eller SMS när du köpte produkten.
  2. Fyll i personnummer eller organisationsnummer.
  3. Klicka på "orderstatus".
Klart, din orderstatus ser du längre ned på sidan.
